# 第2章 問4

$E((X-t)^2)$ を最小とする $t$ を求める。
与式 $= E(t^2 - 2tX + X^2)$
$= t^2 - 2tE(X) + E(X^2)$
$= (t - E(X))^2 - E(X)^2 + E(X^2)$
より $t = E(X)$ となる。

$E(|X-t|)$ を最小とする $t$ を求める。
与式 $= \int_{t}^{\infty} (x-t)f(x)dx + \int_{-\infty}^{t} -(x-t)f(x)dx$
これを $t$ について微分し、式 $= 0$ とする。
まず $\frac{d}{dt} \int_{t}^{\infty} (x-t)f(x)dx = 0 - (t-t)f(t) \cdot 1 + \int_{t}^{\infty} -f(x)dx$
$= 0 - \int_{t}^{\infty} f(x)dx \cdots$ ①

次に $\frac{d}{dt} \int_{-\infty}^{t} -(x-t)f(x)dx = -(t-t)f(t) - 0 + \int_{-\infty}^{t} f(x)dx$
$= 0 + \int_{-\infty}^{t} f(x)dx \cdots$ ②

① $+$ ② $= 0$ なので $\int_{t}^{\infty} f(x)dx = \int_{-\infty}^{t} f(x)dx$ より

よって $x_{center}$ を $X$ の中央値 $X_{center}$ の実現値とすると $t = x_{center}$ となる。 $\blacksquare$
