# 隨ｬ2遶 蝠・

$E((X-t)^2)$ 繧呈怙蟆上→縺吶ｋ $t$ 繧呈ｱゅａ繧九・荳主ｼ・$= E(t^2 - 2tX + X^2)$
$= t^2 - 2tE(X) + E(X^2)$
$= (t - E(X))^2 - E(X)^2 + E(X^2)$
繧医ｊ $t = E(X)$ 縺ｨ縺ｪ繧九・
$E(|X-t|)$ 繧呈怙蟆上→縺吶ｋ $t$ 繧呈ｱゅａ繧九・荳主ｼ・$= \int_{t}^{\infty} (x-t)f(x)dx + \int_{-\infty}^{t} -(x-t)f(x)dx$
縺薙ｌ繧・$t$ 縺ｫ縺､縺・※蠕ｮ蛻・＠縲∝ｼ・$= 0$ 縺ｨ縺吶ｋ縲・縺ｾ縺・$\frac{d}{dt} \int_{t}^{\infty} (x-t)f(x)dx = 0 - (t-t)f(t) \cdot 1 + \int_{t}^{\infty} -f(x)dx$
$= 0 - \int_{t}^{\infty} f(x)dx \cdots$ 竭

谺｡縺ｫ $\frac{d}{dt} \int_{-\infty}^{t} -(x-t)f(x)dx = -(t-t)f(t) - 0 + \int_{-\infty}^{t} f(x)dx$
$= 0 + \int_{-\infty}^{t} f(x)dx \cdots$ 竭｡

竭 $+$ 竭｡ $= 0$ 縺ｪ縺ｮ縺ｧ $\int_{t}^{\infty} f(x)dx = \int_{-\infty}^{t} f(x)dx$ 繧医ｊ
