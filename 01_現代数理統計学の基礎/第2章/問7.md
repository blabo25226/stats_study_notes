# 第2章 問7

## (1)
$E(X) = \int_{0}^{\infty} \{1 - F(x)\}dx - \int_{-\infty}^{0} F(x)dx$, $X$は連続
これを示す。

$E(X) = \int_{-\infty}^{\infty} xf(x)dx$
$= \int_{0}^{\infty} xf(x)dx + \int_{-\infty}^{0} xf(x)dx$
$= \int_{0}^{\infty} \int_{0}^{x} f(x) dt dx - \int_{-\infty}^{0} \int_{x}^{0} f(x) dt dx$

第1項について
$0 < t < x < \infty$ より、積分順序を交換すると
$\int_{0}^{\infty} \int_{t}^{\infty} f(x) dx dt$ となる。

第2項について
$-\infty < x < t < 0$ より、積分順序を交換すると
$\int_{-\infty}^{0} \int_{-\infty}^{t} f(x) dx dt$ となる。

よって
$E(X) = \int_{0}^{\infty} \int_{t}^{\infty} f(x) dx dt - \int_{-\infty}^{0} \int_{-\infty}^{t} f(x) dx dt$
$= \int_{0}^{\infty} [F(x)]_{t}^{\infty} dt - \int_{-\infty}^{0} [F(x)]_{-\infty}^{t} dt$
$= \int_{0}^{\infty} \{1 - F(t)\} dt - \int_{-\infty}^{0} \{F(t) - 0\} dt$

$t$と$x$を入れ替えると
$E(X) = \int_{0}^{\infty} \{1 - F(x)\}dx - \int_{-\infty}^{0} F(x)dx$ となった。 $\blacksquare$

## (2)
$E(X) = \int_{0}^{1} F^{-1}(t)dt$, $X$は連続
これを示す。

右辺 $= \int_{0}^{1} F^{-1}(t)dt$
$y = F^{-1}(t)$ とおくと $F(y) = t$ となる。

積分の範囲と変数は
$t: 0 \to 1$ のとき、$y: -\infty \to \infty$
$\frac{dt}{dy} = f(y)$ すなわち $dt = f(y)dy$
より

右辺 $= \int_{-\infty}^{\infty} y \cdot f(y)dy$ 
これは$E(X)$の定義である $E(X) = \int_{-\infty}^{\infty} xf(x)dx$ に一致する。 $\blacksquare$
