# 隨ｬ2遶 蝠・

## (1)
$f(x) = c x^3, \quad 0 < x < 2$
$\int_0^2 c x^3 dx = 1$ 繧医ｊ
蟾ｦ霎ｺ = $c \left[ \frac{1}{4} x^4 \right]_0^2$
     = $c \left( \frac{1}{4} \cdot 16 - 0 \right)$
     = $4c$
$4c = 1$ 繧医ｊ $c = \frac{1}{4}$ 縺ｨ縺ｪ繧九・
$F(x) = \int_0^x \frac{1}{4} t^3 dt$
     = $\frac{1}{4} \left[ \frac{1}{4} t^4 \right]_0^x$
     = $\frac{1}{16} (x^4 - 0)$
     = $\frac{1}{16} x^4$

## (2)
$f(x) = c e^{-|x|}, \quad x \in \mathbb{R}$
$\int_{-\infty}^\infty c e^{-|x|} dx = 1$ 繧医ｊ
蟾ｦ霎ｺ = $c \int_0^\infty 2 e^{-x} dx$
     = $2c \left[ -e^{-x} \right]_0^\infty$
     = $-2c (0 - 1)$
     = $2c$
$2c = 1$ 繧医ｊ $c = \frac{1}{2}$ 縺ｨ縺ｪ繧九・
$F(x) = \int_{-\infty}^x \frac{1}{2} e^{-|t|} dt$
$x \ge 0$ 縺ｮ縺ｨ縺・$F(x) = \int_{-\infty}^0 \frac{1}{2} e^t dt + \int_0^x \frac{1}{2} e^{-t} dt$
     = $\frac{1}{2} \left[ e^t \right]_{-\infty}^0 + \frac{1}{2} \left[ -e^{-t} \right]_0^x$
     = $\frac{1}{2} (1 - 0) - \frac{1}{2} (e^{-x} - 1)$
     = $1 - \frac{1}{2} e^{-x}$

$x < 0$ 縺ｮ縺ｨ縺・$F(x) = \int_{-\infty}^x \frac{1}{2} e^t dt$
     = $\frac{1}{2} \left[ e^t \right]_{-\infty}^x$
     = $\frac{1}{2} (e^x - 0)$
     = $\frac{1}{2} e^x$

縺励◆縺後▲縺ｦ
$F(x) = \begin{cases} 1 - \frac{1}{2} e^{-x}, & x \ge 0 \\ \frac{1}{2} e^x, & x < 0 \end{cases}$

## (3)
$f(x) = c e^{-2x}, \quad x > 0$
$\int_0^\infty c e^{-2x} dx = 1$ 繧医ｊ
蟾ｦ霎ｺ = $c \left[ -\frac{1}{2} e^{-2x} \right]_0^\infty$
     = $-\frac{1}{2} c (0 - 1)$
     = $\frac{1}{2} c$
$\frac{1}{2} c = 1$ 繧医ｊ $c = 2$

$F(x) = \int_0^x 2 e^{-2t} dt$
     = $2 \left[ -\frac{1}{2} e^{-2t} \right]_0^x$
     = $- (e^{-2x} - 1)$
     = $1 - e^{-2x}$

## (4)
$f(x) = c e^{-x} \exp(-e^{-x}), \quad x > 0$
$\int_0^\infty c e^{-x} \exp(-e^{-x}) dx = 1$ 繧医ｊ
蟾ｦ霎ｺ = $c \int_0^\infty (-e^{-x})' \exp(-e^{-x}) dx$
     = $c \left[ \exp(-e^{-x}) \right]_0^\infty$
     = $c (\exp(0) - \exp(-1))$
     = $c \left( 1 - \frac{1}{e} \right)$
$c \left( 1 - \frac{1}{e} \right) = 1$ 繧医ｊ $c = \frac{1}{1 - \frac{1}{e}} = \frac{e}{e - 1}$

$F(x) = \int_0^x \frac{e}{e-1} e^{-t} \exp(-e^{-t}) dt$
     = $\frac{e}{e-1} \left[ \exp(-e^{-t}) \right]_0^x$
     = $\frac{e}{e-1} (\exp(-e^{-x}) - \exp(-1))$
     = $\frac{e}{e-1} \left( \exp(-e^{-x}) - \frac{1}{e} \right)$
     = $\frac{e}{e-1} \exp(-e^{-x}) - \frac{1}{e-1}$
